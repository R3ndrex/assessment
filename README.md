# Front-end JS engineer test assessment - the React component library

## Project Setup

Go to app folder

```bash
  cd my-vite-app
```

Install all project dependencies

```bash
  npm install
```

Start the React development server

```bash
  npm run dev
```

Start Storybook in development mode

```bash
  npm run storybook
```
